## 3단계: 홈페이지 (src/app/page.tsx) 리뉴얼

현재 홈페이지가 크리에이터 대상 문구로 되어 있음. 소상공인 대상으로 전면 변경:

### Hero 섹션
- 헤드라인: "사장님, 브랜드 굿즈 직접 만들어보세요"
- 서브: "AI가 디자인하고, 굿쯔가 만들어드립니다. 소량도 OK!"
- CTA: "무료로 시작하기"

### 카테고리 섹션 (Phase 1 MVP)
- 명함 (100장 15,000원~)
- 스티커 (50장 10,000원~)
- 전단지/리플렛 (100장 20,000원~)
각 카테고리 카드에 실물 목업 이미지

### 사용 흐름 섹션
1. "카테고리 선택" → 2. "AI가 디자인" → 3. "수량 선택 & 결제" → 4. "배송 완료"
아이콘 + 간단한 설명

### 왜 굿쯔? 섹션
- "디자이너 없이" — AI가 브랜드에 맞는 디자인 제안
- "소량 주문 OK" — 100장부터 가능, 부담 없는 가격
- "빠른 배송" — 주문 후 2~3일 내 도착
- "브랜드 일관성" — 로고/컬러 등록하면 모든 굿즈에 자동 적용

### 후기/신뢰 섹션
- 사장님 후기 카드 (일단 목업 데이터)
- "1,000+ 사장님이 선택했습니다" (일단 목표 수치)

### Footer
- GOODZZ(굿쯔)
- 이메일: support@goodzz.co.kr
- 사업자 정보

## 5단계: 상품 카테고리 구조 변경

현재 상품 구조를 소상공인 특화로 재편:

### 기본 카테고리 (Firestore products 컬렉션 또는 하드코딩)
```
1. 명함 (business-card) — 100/200/500장
2. 스티커 (sticker) — 50/100/200장
3. 전단지 (flyer) — 100/200/500장
4. 에코백 (eco-bag) — 10/20/50개 [Phase 2]
5. 쇼핑백 (shopping-bag) — 50/100개 [Phase 2]
6. 포장박스 (packaging) — 50/100개 [Phase 2]
```

- 카테고리 필터 탭
- 각 상품 카드에 "최소 OO장부터" 표시
