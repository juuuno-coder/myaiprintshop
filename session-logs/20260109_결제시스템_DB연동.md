#### 2. Firebase 상품/주문 데이터 서비스 ✅

**생성된 파일:**
| 파일 | 설명 |
|------|------|
| `src/lib/products.ts` | 상품 CRUD 서비스 (Firestore 연동) |
| `src/lib/orders.ts` | 주문 관리 서비스 (Firestore 연동) |
| `src/app/api/products/route.ts` | 상품 조회 API (카테고리, 검색, 베스트/신상품) |
| `src/app/api/orders/route.ts` | 주문 조회/수정 API |

**상품 API 기능:**

- `GET /api/products` - 전체 상품 조회
- `GET /api/products?category=goods` - 카테고리별 조회
- `GET /api/products?type=best` - 베스트 상품 조회
- `GET /api/products?type=new` - 신상품 조회
- `GET /api/products?q=검색어` - 상품 검색

**주문 API 기능:**

- `GET /api/orders` - 전체 주문 조회 (관리자)
- `GET /api/orders?orderId=xxx` - 단일 주문 조회
- `GET /api/orders?userId=xxx` - 사용자 주문 조회
- `GET /api/orders?stats=true` - 주문 통계 조회
- `PATCH /api/orders` - 주문 상태/배송 정보 업데이트
