## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 11,360줄 (TypeScript 기준) — 중규모 프로젝트
- **주요 기능**: AI 이미지 생성, Fabric.js 에디터, 멀티벤더 시스템, 정산 자동화 완성
- **기술 스택**: Next.js 16, Firebase, PortOne 결제, Gemini AI
- **상태**: 프로덕션 준비 단계, 기본 기능 80% 완성도

### 매출/사용자 현황 (추정)
- **배포 상태**: 도메인 예약 (myaiprintshop.co.kr) — 아직 라이브 트래픽 없음
- **사용자**: 0 (베타 단계)
- **매출**: 0 KRW (검증 전)

### 핵심 과제
1. **제품-시장 적합성 부재** — 타겟 고객 검증 안됨
2. **GTM 전략 부재** — 첫 사용자 획득 경로 미정
3. **경쟁 우위 모호함** — AI 굿즈 시장에서의 차별화 불분명
4. **운영 비용 구조 미파악** — 인쇄/배송 협력사 구축 안됨

| 항목 | 내용 |
|------|------|
| **Vision** | "AI와 창의성으로 누구나 쉽게 맞춤 굿즈를 만들고 판매할 수 있는 세상" |
| **Market Segments** | 직군별: (1) 개인 크리에이터/판매자, (2) 소상공인 (로고 인쇄), (3) 기업 굿즈/이벤트 담당자 |
| **Value Proposition** | "5분 안에 AI로 디자인 생성 → 바로 판매" vs 기존 "2주 이상 디자인 제작료 + 인쇄비" |
| **Trade-offs** | 고려하는 것: 디자인 자유도, 빠른 제작 / 포기하는 것: 대량 주문 할인, 프리미엄 인쇄재질 선택 |
| **Key Metrics** | 가입 → AI 생성 → 장바구니 → 결제 전환율. 초기: 가입자/AI 사용률, 후기: LTV/COGS |
| **Growth** | PLG 90% (AI 체험 무료 제공 → 구매 유도), SLG 10% (블로거/인플루언서 제휴) |
| **Capabilities** | 필수: AI 이미지 생성(Gemini) ✅, 에디터(Fabric.js) ✅, 결제(PortOne) ✅ / 보유 안함: 인쇄 파트너십, 물류 |

| **Cost Structure** | 고정: 개발/운영(月 $5k), 변동: AI 생성 비용(~₩200/회), 인쇄/배송(원가 30-50%) |
| **Revenue Streams** | (1) 주문 마진(상품가-인쇄비-배송비-수수료 10-20%), (2) 벤더 수수료(판매액 5-10%) |

## 3. Beachhead 분석

**Primary Segment: 개인 크리에이터 & 사이드 비즈니스 운영자**

| 기준 | 평가 | 설명 |
|------|------|------|
| **Burning Pain** | ⚠️ 중간 | "디자인 없이 굿즈 판매하고 싶다"는 절실함 있으나, 기존 Printful/Shopify 연동으로 이미 부분 해결 |
| **Willingness to Pay** | ✅ 높음 | 한 건에 ₩10k-50k 주문 × 월 5-10건 = 월 ₩50-500k 잠재 LTV. 크리에이터는 소득 창출에 민감 → 전환율 예상 15-20% |
| **Winnability** | ⚠️ 낮음 | 이미 Printful/Merch by Amazon 등 강자 점유. 한국 시장 특화도 약함 (배송비/A/S 문제) |
| **Referral Potential** | ✅ 높음 | 크리에이터 A가 성공 → SNS 공유 → 다른 크리에이터 유입. 바이럴 루프 가능성 있음 |

**2순위 Segment: 소상공인 (떡케이크/의류샵/카페)**
- 현수막/스티커/에코백 필요하지만 최소주문량(MOQ) 높음 → 제약

## 4. Growth Loop

**Adoption Growth Loop (가장 적합)**

```
신규 가입 → 무료 AI 생성 체험 (10회)
  ↓
디자인 완성 → 5분 내 상품 미리보기
  ↓
주문 (₩9,900 최소) → 실제 상품 수령
  ↓
SNS 공유 (#내가_만든_굿즈) → 친구 방문
  ↓
다음 주문 or 벤더 가입 (정산 시스템 활용)
```

**핵심 메트릭**
1. AI 생성 완료율: 가입자 중 60% 이상 1회 이상 생성
2. 첫 주문 전환율: 디자인 생성자 중 15% 이상 주문
3. 리피트율: 첫 구매자 중 30% 이상 3개월 내 재주문
4. 벤더 가입: 월 매출 ₩1m 이상 매출처에서 5% 이상 벤더 전환

## 5. 가격 전략

### 현재 미정
프로토콜 상 **무료** → 디자인 생성 후 결제 구조로 보임.

### 권장 가격 (Freemium 모델)

| 상품 | 가격 | 이유 |
|------|------|------|
| AI 생성 (무료) | ₩0 × 10회/월 | PLG 드라이버. 디자인 없이 시작 불가 |
| 최소 상품 | ₩9,900 | 심리적 진입장벽 낮춤 (₩10k 미만) |
| 프리미엄 상품 | ₩20k-50k | 에코백/티셔츠 (마진율 30-40%) |
| 벤더 정산 | 5-10% 수수료 | Shopify 기준(2%), 한국 배송비 고려 |
| 프로 플랜 | 월 ₩9,900 (미래) | AI 무제한 + 템플릿 라이브러리 (초기: 제외) |

### 가격 검증 필요
- 첫 100명 사용자 가격 민감도 테스트
- 경쟁사 (국내 굿즈 쇠솥 등) 가격 벤치마킹
- 원가(인쇄/배송) 기반 마진율 재계산

## 7. North Star Metric

### 권장 NSM
**월간 활성 판매자당 평균 주문 수 (Monthly Orders per Active Seller)**

- 정의: 월간 1건 이상 주문을 생성한 판매자의 평균 주문 건수
- 목표: 6개월 내 3건/판매자/월 달성
- 이유: (1) 고객 만족도 대리 지표, (2) 리피트 강도 측정, (3) LTV 개선

### Input Metrics (주간 추적)
1. **신규 가입자 수**: 주 목표 50명 (초기)
2. **AI 생성 완료율**: 가입자 대비 60% 이상
3. **첫 주문 전환율**: 디자인 완성자 대비 15% 이상
4. **리피트율 (D30)**: 첫 구매자 중 30% 이상 재주문
5. **평균 주문가**: ₩20k 이상 (초기는 ₩15k)

## 8. 3단계 로드맵

### Phase 1: MVP 검증 (현재~4월, 1개월)

**목표**: 첫 50명 사용자 확보, 제품-시장 적합성 검증

**실행 항목**
- [ ] 베타 테스터 20명 모집 (커뮤니티/네트워크)
- [ ] 고객 인터뷰 (10명, 각 30분) — "굿즈 판매 경험" 깊이 있게
- [ ] 랜딩페이지 + 대기자 명단 (마케팅 사이트 아님. 최소 1장짜리)
- [ ] 인쇄/배송 1곳 파트너십 확정 (Woopress or 로컬)
- [ ] 프로덕션 배포 (myaiprintshop.co.kr, Vercel)

**성공 지표**
- 대기자 100명 이상
- 베타 테스트 완료율 80% 이상
- 첫 5건 주문 달성

### Phase 2: 초기 트랙션 (4월~6월, 3개월)

**목표**: 월 100건 주문, 벤더 시스템 오픈

**실행 항목**
- [ ] 크리에이터 커뮤니티 타겟 (당근마켓, 오늘의집, 당신의손으로)
- [ ] 인플루언서 파일럿 (5-10명, 각 ₩100k 커미션)
- [ ] 벤더 대시보드 정산 자동화 (이미 코드 준비됨)
- [ ] 고객지원 자동화 (FAQ 챗봇, 이메일 템플릿)
- [ ] A/B 테스트 (가격, AI 프롬프트, 디자인 템플릿)

**성공 지표**
- 월 100건 주문 달성
- CAC ₩5k 이하
- 첫 벤더 ₩1m 매출 달성

### Phase 3: 확장 & 최적화 (6월~12월, 6개월)

**목표**: 월 ₩50m 매출, 5개 쇼핑몰 통합 (Shopify/당근마켓/Naver 등)

**실행 항목**
- [ ] Shopify 임베드 SDK 오픈 (크리에이터 숍 연동)
- [ ] 구독 모델 도입 (AI 무제한 + 프리미엄 템플릿)
- [ ] 인쇄 파트너 2곳 이상 추가 (배송 속도/비용 최적화)
- [ ] 모바일 앱 (React Native, Expo) 출시 검토
- [ ] 해외 시장 테스트 (일본, 동남아)

**성공 지표**
- 월 ₩50m 매출 (목표)
- 활성 판매자 500명 이상
- LTV/CAC 비율 3:1 이상

## 핵심 한 줄 요약

> **"AI로 5분 안에 굿즈 디자인 완성 → 즉시 판매하는 한국식 노코드 이커머스"**

### 현재 위험도: 🔴 높음
- 기술은 준비됐으나 제품-시장 적합성 & GTM 전략 부재
- 첫 100명 사용자 획득 전까지는 실패 가능성 높음
- Phase 1 (1개월)에서 50명 이상 확보 실패 시 재전략 필요

### 투자 결정 기준
1. **양수 신호**: 베타 테스터 80% 완료율 + 첫 5건 주문 + 월 100건 경로 보임
2. **음수 신호**: 커뮤니티 관심 5% 이하 + 전환율 5% 미만 + COGS > 60%
3. **다시 생각해볼 요소**: "AI 굿즈"에서 "AI 디자인 도구"로 피벗 검토 (SaaS 모델)

**최종 조언**: 이 프로젝트는 **기술 역량**은 뛰어나나 **사업 검증**이 부족합니다. 지금 바로 할 일은 코드 작성이 아니라 **고객 인터뷰 10명** + **초간단 랜딩페이지** + **인쇄사 파트너십 1곳**입니다. 2주 내에 이 3가지를 확보하면 Phase 2로 진행할 가치가 있습니다.
