# 🚀 MyAIPrintShop 개발 로드맵

사용자님의 요청에 따라 전체 기능 개발 순서를 정리했습니다.
이 문서를 기준으로 하나씩 순차적으로 개발을 진행하겠습니다.

## 1. 쇼핑 핵심 기능 (Shopping Core)

- [x] **상점 메인 (`/shop`)**: 상품 목록 그리드, 카테고리 탭
- [x] **상품 상세 (`/shop/[id]`)**: AI 디자인 캔버스, 옵션 UI
- [x] **상태 관리 (Store)**: 장바구니/위시리스트 데이터 구조 설계 (Zustand)
- [x] **장바구니 담기 로직**: 상세 페이지에서 옵션 + AI 디자인 이미지 포함하여 담기 구현
- [x] **장바구니 페이지 (`/cart`)**: 담은 상품 목록 확인, 수량 변경, 삭제, 총 금액 계산
- [x] **위시리스트 페이지 (`/wishlist`)**: 찜한 상품 목록 확인 및 장바구니로 이동

## 2. 주문 및 결제 (Checkout & Order)

- [x] **주문서 작성 페이지 (`/checkout`)**:
  - 배송지 입력 폼 (이름, 전화번호, 주소)
  - 주문 상품 요약
- [x] **결제 시뮬레이션**: PG사 연동 대신 '결제하기' 버튼 클릭 시 주문 완료 처리
- [x] **주문 완료 페이지 (`/order/complete`)**: 주문 번호 및 감사 메시지 표시
- [x] **와우프레스 API 연동 (Mock)**: 주문 데이터를 와우프레스 API 규격에 맞춰 전송(하는 척)하는 로직

## 3. 마이페이지 & 고객 편의 (My Page)

- [x] **마이페이지 홈 (`/mypage`)**: 주문/배송 현황 대시보드
- [x] **주문 내역 조회**: 과거 주문 리스트 및 상세 내역(배송조회 버튼 포함)
- [x] **쿠폰함**: 쿠폰 등록 및 보유 쿠폰 리스트 (Mock 데이터)

## 4. 고도화 (Refinement)

- [x] **배송 조회**: 가상의 운송장 번호 연동
- [x] **모바일 최적화 점검**: 모든 페이지 반응형 확인

---

### 📅 진행 방식

**"컨펌"** 해주시면 위 순서대로 하나씩 개발하고 체크박스를 채워나가겠습니다.
자리를 비우신 동안 **1번(쇼핑 핵심) -> 2번(주문)** 까지 쭉 진행해두는 것을 목표로 하겠습니다.
